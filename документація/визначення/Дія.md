# Дія

`Дія` це батьківська структура для всіх дій
Мавки.

Чародії для структури `Дія` та її предметів заборонено перезаписувати.

`Дія` описує такі властивості для своїх предметів:

```мавка
структура Дія
  назва
  параметри
  код
  я
кінець
```

```мавка
дія Дія.чародія_текст()
```

```мавка
дія Дія.чародія_юнікод()
```
