# афс

Асинхронна файлова система.

```мавка
взяти біб афс

афс.записати("привітики", "./якийсь.текст", дія()
  афс.прочитати(Текст, "./якийсь.текст", дія(значення)
    друк(значення)
  кінець)
кінець)
```

## Записати у файл

```мавка
;; записує значення у файл
;; якщо "значення" є Текст, перекодування в юнікод НЕ БУДЕ
;;                          тобто у файл запишеться дієкод
;; якщо "значення" є Дані, буде записано без змін
;; якщо "куди" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія записати(значення, куди, відклик)
```

Приклади:

```мавка
д = Дані.виділити(4)
д[0] = 33
д[1] = 34
д[2] = 35
д[3] = 36

афс.записати(
  д,
  "якісь.дані",
  дія(п): друк(п якщо п інакше "записано!")
)
```

```мавка
афс.записати(
  "привітики",
  "./якийсь.текст",
  дія(п): друк(п якщо п інакше "записано!")
) ;; запишеться дієкод
```

## Прочитати з файлу

```мавка
;; читає весь файл
;; якщо "тип" == Текст, прочитане значення буде вернуто як текст БЕЗ ПЕРЕКОДУВАННЯ
;; якщо "шлях" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія прочитати(тип, шлях, відклик)
```

Приклади:

```мавка
афс.прочитати(
  Дані,
  "./якісь.дані",
  дія(значення, п): друк(п якщо п інакше значення)
)
```

```мавка
афс.прочитати(
  Текст,
  "./якийсь.текст",
  дія(значення, п): друк(п якщо п інакше значення)
) ;; прочитається Текст БЕЗ ПЕРЕКОДУВАННЯ
```

## Дописати у файл

```мавка
;; дописує значення у файл
;; якщо "значення" є Текст, перекодування в юнікод НЕ БУДЕ
;;                          тобто у файл запишеться дієкод
;; якщо "значення" є Дані, буде дописано без змін
;; якщо "куди" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія дописати(значення, куди, відклик)
```

Приклади:

```мавка
д = Дані.виділити(4)
д[0] = 33
д[1] = 34
д[2] = 35
д[3] = 36

афс.дописати(
  д,
  "./якісь.дані",
  дія(п): друк(п якщо п інакше "дописано!")
)
```

```мавка
афс.дописати(
  "привітики",
  "./якийсь.текст",
  дія(п): друк(п якщо п інакше "дописано!")
) ;; доповниться дієкод
```

## Створити папку

```мавка
;; якщо "шлях" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія створити_папку(шлях, відклик)
```

Приклади:

```мавка
афс.створити_папку(
  "./а",
  дія(п): друк(п якщо п інакше "створено!")
)
```

```мавка
афс.створити_папку(
  "./а/б/в",
  дія(п): друк(п якщо п інакше "створено!")
)
```

## Видалити файл/папку

```мавка
;; якщо "шлях" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія видалити(шлях, відклик)
```

Приклади:

```мавка
афс.видалити(
  "./якісь.дані",
  дія(видалено, п): друк(п якщо п інакше видалено)
)
```
