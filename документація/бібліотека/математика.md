# Математика

```мавка
взяти біб М

друк(М.абс(-10))
друк(М.ПІ)
друк(М.корінь(9))
друк(М.степінь(2, 3))
друк(М.округлити(3.14))
друк(М.найбільше([1, 2, 3, 4, 5]))
друк(М.найменше([1, 2, 3, 4, 5]))
```
