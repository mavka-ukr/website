# аввід

Асинхронний ввід.

```мавка
взяти біб аввід

аввід.запитати(
  "Як вас звати? ",
  дія(значення)
    друк("Привіт, %(значення)!")
  кінець
)
```

Дії:

```мавка
;; прочитати зі стандартного вводу
;; повідомлення має бути текстом
;; у відклик передається Текст якщо вдалось перекодувати,
;;                             інакше Дані
дія запитати(повідомлення, відклик)
```
