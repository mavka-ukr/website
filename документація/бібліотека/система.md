# Система

```мавка
взяти біб система

система.записати("якийсь.текст", "привітики")

друк(система.прочитати("якийсь.текст", Текст))
```

## Дії


### Записати у файл

```мавка
;; записує значення у файл
;; якщо "шлях" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
;; якщо "значення" є Текст, перекодування в юнікод НЕ БУДЕ
;;                          тобто у файл запишеться дієкод
;; якщо "значення" є Дані, буде записано без змін
дія записати(шлях, значення)
```

Приклади:

```мавка
система.записати("якийсь.текст", "привітики") ;; запишеться дієкод
```

### Прочитати з файлу

```мавка
;; читає весь файл
;; якщо "шлях" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
;; якщо "тип" == Текст, прочитане значення буде вернуто як текст БЕЗ ПЕРЕКОДУВАННЯ
дія прочитати(шлях, тип)
```

Приклади:

```мавка
значення = система.прочитати("якийсь.текст") ;; вернеться "Дані"
```

```мавка
значення = система.прочитати("якийсь.текст", Текст) ;; вернеться "Текст"
```
