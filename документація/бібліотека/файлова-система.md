# Файлова система

```мавка
взяти біб фс

фс.записати("привітики", "./якийсь.текст")

друк(фс.прочитати(Текст, "./якийсь.текст"))
```

## Записати у файл

```мавка
;; записує значення у файл
;; якщо "значення" є Текст, перекодування в юнікод НЕ БУДЕ
;;                          тобто у файл запишеться дієкод
;; якщо "значення" є Дані, буде записано без змін
;; якщо "куди" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія записати(значення, куди)
```

Приклади:

```мавка
д = Дані.виділити(4)
д[0] = 33
д[1] = 34
д[2] = 35
д[3] = 36

фс.записати(д, "якісь.дані")
```

```мавка
фс.записати("привітики", "./якийсь.текст") ;; запишеться дієкод
```

## Прочитати з файлу

```мавка
;; читає весь файл
;; якщо "тип" == Текст, прочитане значення буде вернуто як текст БЕЗ ПЕРЕКОДУВАННЯ
;; якщо "куди" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія прочитати(тип, куди)
```

Приклади:

```мавка
значення = фс.прочитати(Дані, "./якісь.дані")
```

```мавка
значення = фс.прочитати(Текст, "./якийсь.текст") ;; прочитається Текст БЕЗ ПЕРЕКОДУВАННЯ
```

## Дописати у файл

```мавка
;; дописує значення у файл
;; якщо "значення" є Текст, перекодування в юнікод НЕ БУДЕ
;;                          тобто у файл запишеться дієкод
;; якщо "значення" є Дані, буде дописано без змін
;; якщо "куди" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія дописати(значення, куди)
```

Приклади:

```мавка
д = Дані.виділити(4)
д[0] = 33
д[1] = 34
д[2] = 35
д[3] = 36

фс.дописати(д, "./якісь.дані")
```

```мавка
фс.дописати("привітики", "./якийсь.текст") ;; доповниться дієкод
```

## Створити папку

```мавка
;; якщо "шлях" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія створити_папку(шлях)
```

Приклади:

```мавка
фс.створити_папку("./а")
```

```мавка
фс.створити_папку("./а/б/в")
```

## Видалити файл/папку

```мавка
;; якщо "шлях" є Текст, його буде перекодовано в юнікод (юнікс, віндовс)
дія видалити(шлях)
```

Приклади:

```мавка
фс.видалити("./якісь.дані")
```
