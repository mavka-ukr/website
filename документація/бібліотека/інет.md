# інет

Інтернет (_експериментально_).

## Запустити _TCP(eng)_ слугу

```мавка
взяти біб інет

дія обробник_запуску(слуга, помилка)
кінець

дія обробник_помилки(слуга, помилка)
кінець

дія обробник_зупинки(слуга)
кінець

дія обробник_підключення_звʼязку(звʼязок, слуга)
кінець

дія обробник_даних_звʼязку(звʼязок, дані, слуга)
кінець

дія обробник_стікання_звʼязку(звʼязок, слуга)
кінець

дія обробник_закінчення_звʼязку(звʼязок, слуга)
кінець

дія обробник_помилки_звʼязку(звʼязок, помилка, слуга)
кінець

дія обробник_відключення_звʼязку(звʼязок, слуга)
кінець

інет.служити(
  іа="0.0.0.0",
  порт=8080,
  формат=Дані, ;; може бути Дані або Текст

  обробник_запуску=обробник_запуску,
  обробник_помилки=обробник_помилки,
  обробник_зупинки=обробник_зупинки,

  обробник_підключення_звʼязку=обробник_підключення_звʼязку,
  обробник_даних_звʼязку=обробник_даних_звʼязку,
  обробник_стікання_звʼязку=обробник_стікання_звʼязку,
  обробник_закінчення_звʼязку=обробник_закінчення_звʼязку,
  обробник_помилки_звʼязку=обробник_помилки_звʼязку,
  обробник_відключення_звʼязку=обробник_відключення_звʼязку,
)
```

## Підключитись до _TCP(eng)_ слуги

```мавка
взяти біб інет

дія обробник_підключення(звʼязок, помилка)
кінець

дія обробник_даних(звʼязок, дані)
кінець

дія обробник_стікання(звʼязок)
кінець

дія обробник_закінчення(звʼязок)
кінець

дія обробник_помилки(звʼязок, помилка)
кінець

дія обробник_відключення(звʼязок)
кінець

інет.підключитись(
  іа="0.0.0.0",
  порт=8080,
  формат=Дані, ;; може бути Дані або Текст

  обробник_підключення=обробник_підключення,
  обробник_даних=обробник_даних,
  обробник_стікання=обробник_стікання,
  обробник_закінчення=обробник_закінчення,
  обробник_помилки=обробник_помилки,
  обробник_відключення=обробник_відключення,
)
```

## Методи класу `Звʼязок`

```мавка
;; надсилає повідомлення
;; значення має бути даними або текстом
;; якщо вернулось "недійсне", тоді треба почекати на "обробник_стікання" і в ньому ще раз надіслати
дія надіслати(значення)
```

```мавка
;; повідомляє про закінчення надсилання (FIN)
;; значення має бути даними або текстом
дія закінчити(значення)
```

```мавка
;; відключає і знищує звʼязок без очікування
дія знищити()
```

## Методи класу `Слуга`

```мавка
;; зупиняє слугу і відключає клієнтів за необхідності
дія зупинити(закрити_підключення)
```
