# ПроміжокЧисел

```мавка
клас ПроміжокЧисел
  від ;; завжди включно
  до ;; включно лише коли властивість "включно" дійсна
  включно
кінець
```

```мавка
дія ПроміжокЧисел.чародія(від, до, включно)
```

```мавка
дія ПроміжокЧисел.чародія_перебір()
```

```мавка
ПроміжокЧисел.Перебір ;; клас перебору проміжку чисел
```

---

```мавка
дч = 0..5 ;; ПроміжокЧисел(0, 5)

дч1 = 0..=5 ;; ПроміжокЧисел(0, 5, дійсне)

;; ...тощо
```

перебрати 0..5 як п
  друк(п)
кінець
