---

```мавка
;; створити список з тексту
сп = Список("привіт")
```

```мавка
;; створити список з послідовності
сп = Список(Послідовність("привіт"))
```

```мавка
сп = ["Привіт", 'Д', 3.14]

;; клонувати список
сп1 = Список(сп)
```

```мавка
;; отримати елемент списку
друк(сп[18])
```

```мавка
;; змінити елемент списку
сп[18] = "привіт"
```

```мавка
;; перебрати список
перебрати сп як ел
  друк(ел)
```
