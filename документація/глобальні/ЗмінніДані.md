# ЗмінніДані

`ЗмінніДані` це структура, предмети якої зберігають змінні сирі дані у вигляді байтів.

```мавка
структура ЗмінніДані
  розмір
  елементи
кінець
```

```мавка
дія ЗмінніДані.чародія_отримати_елемент(ключ)
```

```мавка
дія ЗмінніДані.чародія_змінити_елемент(ключ, значення)
```

```мавка
дія ЗмінніДані.чародія_перебір()
```

```мавка
дія ЗмінніДані.чародія_текст()
```
