# 孝械泻褋褌

`孝械泻褋褌` 胁懈泻芯褉懈褋褌芯胁褍褦 [袛褨褦泻芯写](./写褨褦泻芯写.html).

```屑邪胁泻邪
泻谢邪褋 孝械泻褋褌
  写芯胁卸懈薪邪
泻褨薪械褑褜
```

```屑邪胁泻邪
写褨褟 孝械泻褋褌.褔邪褉芯写褨褟_褔懈褋谢芯()
```

```屑邪胁泻邪
写褨褟 孝械泻褋褌.褔邪褉芯写褨褟_褌械泻褋褌()
```

```屑邪胁泻邪
写褨褟 孝械泻褋褌.褔邪褉芯写褨褟_锌械褉械斜褨褉()
```

```屑邪胁泻邪
;; 胁械褉褌邪褦 蟹褉褨蟹 褌械泻褋褌褍 蟹 锌芯蟹懈褑褨褩 "胁褨写" (胁泻谢褞褔薪芯) 写芯 锌芯蟹懈褑褨褩 "写芯" (薪械 胁泻谢褞褔薪芯)
写褨褟 孝械泻褋褌.蟹褉褨蟹(胁褨写, 写芯)
```

```屑邪胁泻邪
孝械泻褋褌.袩械褉械斜褨褉 ;; 泻谢邪褋 锌械褉械斜芯褉褍 褌械泻褋褌褍
```
