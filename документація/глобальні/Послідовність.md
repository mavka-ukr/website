---

```мавка
;; створити пусту послідовність
пс = ()
```

```мавка
;; створити послідовність з одним елементом
пс = ("привіт",)
```

```мавка
;; створити послідовність
пс = (1, "2", 3.14, дійсне)
```

```мавка
;; створити послідовність з тексту
пс = Послідовність("привіт")
```

```мавка
;; створити послідовність зі списку
пс = Послідовність(["Привіт", 'Д', 3.14])
```

```мавка
;; отримати елемент послідовності
друк(пс[18])
```

```мавка
;; перебрати послідовність
перебрати пс як ел
  друк(ел)
кінець
```
