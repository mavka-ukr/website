# Послідовність

```мавка
клас Послідовність
  розмір
кінець
```

```мавка
дія Послідовність.чародія_перебір()
```

```мавка
Послідовність.Перебір ;; клас перебору послідовності
```

---

```мавка
;; створити послідовність з тексту
пс = Послідовність("привіт")
```

```мавка
;; створити послідовність зі списку
пс = Послідовність(["Привіт", 'Д', 3.14])
```

```мавка
;; отримати елемент послідовності
друк(пс[18])
```

```мавка
;; перебрати послідовність
перебрати пс як ел
  друк(ел)
кінець
```
