# Дані

`Дані` це клас предметів які зберігають сирі дані у вигляді байтів.

```мавка
клас Дані
  розмір
кінець
```

```мавка
дія Дані.чародія_текст()
```

```мавка
дія Дані.чародія_перебір()
```

```мавка
Дані.Перебір ;; клас перебору даних
```

```мавка
Дані.виділити(розмір)
```

```мавка
Дані.обнулено(розмір)
```

---

```мавка
;; виділити дані вказаного розміру
д = Дані.виділити(128)
```

```мавка
;; виділити чисті (заповнені нулями) дані вказаного розміру
д = Дані.обнулено(128)
```

```мавка
;; створити дані з тексту
д = Дані("привіт")
```

```мавка
;; створити дані з послідовності
д = Дані(Послідовність("привіт"))
```

```мавка
;; створити дані зі списку
д = Дані(['П', 'р', 'и', 'в', 'і', 'т']) ;; Дані([46, 79, 70, 83, 71, 81])
```

```мавка
д = Дані("привіт")

;; клонувати дані
д1 = Дані(д)
```

```мавка
;; отримати елемент даних
друк(д[18])
```

```мавка
;; змінити елемент даних
д[18] = 122
```

```мавка
;; перебрати дані
перебрати д як б
  друк(б)
кінець
```
