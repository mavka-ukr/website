# Дані

`Дані` це структура, предмети якої зберігають сирі дані у вигляді байтів.

```мавка
структура Дані
  розмір
кінець
```

```мавка
дія Дані.чародія_перебір()
```

```мавка
;; виділити дані довільного розміру
д = Дані(128)
```

```мавка
;; отримати елемент даних
друк(д[18])
```

```мавка
;; змінити елемент даних
д[18] = 122
```

```мавка
;; перебрати дані
перебрати д як б
  друк(б)
кінець
```
