# Дії порівняння

Мавка дозволяє записувати виконання основних порівняльних дій над предметами.

Рівно:

```мавка
а == б
```

```мавка
а рівно б
```

Не рівно:

```мавка
а != б
```

```мавка
а не рівно б
```

Більше (лише для чисел):

```мавка
а > б
```

```мавка
а більше б
```

Менше (лише для чисел):

```мавка
а < б
```

```мавка
а менше б
```

Не більше (лише для чисел):

```мавка
а <= б
```

```мавка
а не більше б
```

Не менше (лише для чисел):

```мавка
а >= б
```

```мавка
а не менше б
```

Містить (лише для списків та словників):

```мавка
а містить б
```

Не містить:

```мавка
а не містить б
```
