# Дії порівняння

<subject>Мавка</subject> дозволяє записувати виконання основних порівняльних <keyword>дій</keyword> над обʼєктами.

Рівно:

```мавка
а == б
```

```мавка
а рівно б
```

Не рівно:

```мавка
а != б
```

```мавка
а не рівно б
```

Більше:

```мавка
а > б
```

```мавка
а більше б
```

Менше:

```мавка
а < б
```

```мавка
а менше б
```

Не більше:

```мавка
а <= б
```

```мавка
а не більше б
```

Не менше:

```мавка
а >= б
```

```мавка
а не менше б
```
