# Дії порівняння

Запис порівняльних дій над предметами відбувається так:

Рівно:

```мавка
а == б
```

```мавка
а рівно б
```

Не рівно:

```мавка
а != б
```

```мавка
а не рівно б
```

Більше (лише для чисел):

```мавка
а > б
```

```мавка
а більше б
```

Менше (лише для чисел):

```мавка
а < б
```

```мавка
а менше б
```

Не більше (лише для чисел):

```мавка
а <= б
```

```мавка
а не більше б
```

Не менше (лише для чисел):

```мавка
а >= б
```

```мавка
а не менше б
```

Є (перевірити структуру предмета):

```мавка
а є Число
```

Не є:

```мавка
а не є Молекула
```
