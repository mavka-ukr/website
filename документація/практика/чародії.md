# Чародії

## `чародія`

Дія `чародія` здійснюється (якщо визначено) під час створення предмета зі структури.

```мавка
структура Ракета
  назва
кінець

дія Ракета.чародія(назва)
  я.назва = "Ракета %(назва)"
кінець

ракета = Ракета("Реакція")

друк(ракета) ;; Ракета(назва="Ракета Реакція")
```

## `чародія_перебір`

Дія `чародія_перебір` здійснюється для отримання перебору.

```мавка
структура Машини
  елементи
кінець

дія Машини.чародія_перебір()
  ;; тут необхідно вернути перебір
кінець

перебрати машини як м
  ;; ...
кінець
```

## `чародія_діапазон`

Дія `чародія_діапазон` здійснюється для отримання діапазону.

```мавка
діапазон = 0..=10

;; те саме що

діапазон = 0.чародія_діапазон(10, включно=дійсне)
```

## `чародія_ціле`

Дія `чародія_ціле` здійснюється для перетворення значення на ціле число.

```мавка
друк(Ціле.з("123") + 1) ;; 124
```

## `чародія_дробове`

Дія `чародія_дробове` здійснюється для перетворення значення на дробове число.

```мавка
друк(Ціле.з("123.4") + 0.5) ;; 123.9
```

## `чародія_текст`

Дія `чародія_текст` здійснюється для перетворення значення на текст.

```мавка
друк(Текст.з(123.456)) ;; 123.456
```
