# Арифметичні дії

<subject>Мавка</subject> дозволяє записувати виконання основних арифметичних дій над обʼєктами.

Додати:

```мавка
а + б
```

Відняти:

```мавка
а - б
```

Помножити:

```мавка
а * б
```

Поділити:

```мавка
а / б
```

Отримати неповну частку ділення:

```мавка
а // б
```

Отримати остачу ділення:

```мавка
а % б
```

Піднести до степеня:

```мавка
а ** б
```
