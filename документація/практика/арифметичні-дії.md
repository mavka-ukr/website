# Арифметичні дії

<subject>Мавка</subject> дозволяє записувати виконання основних арифметичних <keyword>дій</keyword> над обʼєктами.

Додати:

```мавка
а + б
```

Відняти:

```мавка
а - б
```

Помножити:

```мавка
а * б
```

Поділити:

```мавка
а / б
```

Піднести до степеня:

```мавка
а ** б
```

Поділити націло:

```мавка
а // б
```

Знайти остачу від ділення:

```мавка
а % б
```
