# Праця з паками

Паки зберігаються в теці `паки` в корені проекту:

```
паки/
├─ привіт/
│  ├─ привіт.м
старт.м
```

Головний <keyword>Модуль<keyword> `привіт.м` пака:

```мавка
дія привіт()
  друк("Привіт від Лесі!")
кінець

дати привіт
```

`старт.м`:

```мавка
взяти пак привіт

привіт.привіт()
```
