# Зовнішні модулі

Взяти зовнішній модуль проєкту:

```мавка
взяти модуль привіт

привіт.привіт()
```

```мавка
взяти модуль привіт як суперпривіт

суперпривіт.привіт()
```

```мавка
взяти модуль привіт [привіт]

привіт()
```

```мавка
взяти модуль привіт [привіт як суперпривіт]

суперпривіт()
```

```мавка
взяти модуль назва_підпапки.привіт

друк(привіт)
```
