# Зовнішні модулі

Взяти зовнішній модуль проєкту:

```мавка
взяти привіт

привіт.привіт()
```

```мавка
взяти привіт як суперпривіт

суперпривіт.привіт()
```

```мавка
взяти привіт [привіт]

привіт()
```

```мавка
взяти привіт [привіт як суперпривіт]

суперпривіт()
```

```мавка
взяти назва_підпапки.інший_модуль

друк(інший_модуль)
```
