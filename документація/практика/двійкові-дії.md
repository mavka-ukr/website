# Двійкові дії

<subject>Мавка</subject> дозволяє записувати виконання основних двійкових дій над предметами.

Зсув вліво:

```мавка
а << б
```

Зсув вправо:

```мавка
а >> б
```

Зсув вправо зі заповненням:

```мавка
а >>> б
```

Двійкове І:

```мавка
а & б
```

Двійкове АБО:

```мавка
а | б
```

Двійкове виключне АБО:

```мавка
а ^ б
```

Двійкове НЕ:

```мавка
~а
```
