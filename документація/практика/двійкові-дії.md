# Двійкові дії

Мавка дозволяє записувати виконання основних двійкових дій над предметами числа.

Зсув вліво:

```мавка
а << б
```

Зсув вправо:

```мавка
а >> б
```

Зсув вправо 2:

```мавка
а >>> б
```

Двійкове І:

```мавка
а ∧ б
```

Двійкове АБО:

```мавка
а ∨ б
```

Двійкове виключне АБО:

```мавка
а вабо б
```

Двійкове НЕ:

```мавка
¬а
```
