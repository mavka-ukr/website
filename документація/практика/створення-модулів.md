# Створення модулів

Записати створення модуля в Мавці можна так:

```мавка
модуль
  якийсь_предмет = 2.10

  дати якийсь_предмет
кінець
```

Якщо додати назву до цього запису, то після створення, модуль буде відразу визначено в поточному середовищі:

```мавка
модуль якийсь_модуль
  якийсь_предмет = 2.10

  дати якийсь_предмет
кінець
```
