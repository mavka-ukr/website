# Створення модулів

Записати створення модуля в <subject>Мавці</subject> можна так:

```мавка
модуль
  якийсь_обʼєкт = 2.10

  дати якийсь_обʼєкт
кінець
```

Якщо додати назву до цього запису, то після створення буде відразу визначено субʼєкт цього модуля:

```мавка
модуль якийсь_модуль
  якийсь_обʼєкт = 2.10

  дати якийсь_обʼєкт
кінець
```
