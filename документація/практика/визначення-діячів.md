# Визначення діячів

Вручну визначити або перевизначити діяча в Мавці можна використовуючи знак `=`:

```мавка
кіт_моєї_сестри = Кіт("Тимофій")
```

```мавка
число_пі_наближено = 3.14159
дві_третини = 2 / 3
```

```мавка
м = 5.97219 * 10 ** 24
шс = 299792458
Е = м * шс ** 2
```

Звернутись до діяча можна з поточної дійсності або із зовнішньої:

```мавка
а = 2

дія ф()
  б = 3

  друк(а) ;; 2
кінець

ф()

друк(а) ;; 2

друк(б) ;; помилка
```

Існують також допоміжні оператори `+=`, `-=`, `*=`, `/=`, `%=`, `**=` що
дозволяють простіше перевизначити діяча в поточній дійсності із застосуванням до нього арифметичних дій:

```мавка
а = 2

а += 3 ;; те саме що а = а + 3

друк(а) ;; 5
```
