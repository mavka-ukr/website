# Розмова

Розмова Мавки дозволяє працювати з Мавкою в режимі розмови. Ви пишете інструкцію,
Машина Мавки її виконує й відповідає.

Для цього потрібно запускати Мавку без аргументів:

```shell
мавка
```

```plaintext
Розмова Мавки {{ВЕРСІЯ_МАВКИ}}
- сума = дія(а, б): а + б
недійсне
- сума(2, 3)
5
```
