# Здійснення дій

Записати здійснення в Мавці можна так:

```мавка
додати(2, 2)
```

Тобто ліворуч має бути значення, а праворуч в дужках задіяні:

```мавка
а.б.в("привіт")

с[0](123)

с[0](1)(2)(3)

;; ...тощо
```

Існує також можливість назвати задіяні:

```мавка
додати(а=2, б=2)
```

Позиційні задіяні та названі задіяні можна комбінувати, але названі задіяні завжди повинні йти після позиційних:

```мавка
додати(2, б=2)
```
