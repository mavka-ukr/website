# Створення словників

Записати створення словника в Мавці можна так:

```мавка
{}
```

```мавка
{
  "істина" => "є",
  "свідомість" => "знання",
  "організація" => "діячі",
  "тридцять" => 30,
  40 => "сорок",
  50 => 20 + 20
}
```

Отримати елемент словника:

```мавка
сл["істина"]

сл[50]
```

Змінити елемент словника:

```мавка
сл["істина"] = "єсть"

сл[50] = 30 + 20
```
