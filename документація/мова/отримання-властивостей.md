# Отримання властивостей

Отримати властивість предмета в Мавці можна так:

```мавка
а.б
```

```мавка
а.б.г
```

Тобто ліворуч має бути значення, а праворуч після крапки назва властивості:

```мавка
а.б[г].д().е ;; ...тощо
```

```мавка
клас Потяг
  назва
кінець

пв = Потяг("Вітер")

друк(пв.назва)
```
