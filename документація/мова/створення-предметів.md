# Створення предметів

Записати створення предмета класу можна так само як здійснення
дії:

```мавка
Молекула(атоми=[...])
```

```мавка
Книжка(
  назва="Лісова пісня",
  автор=Автор(псевдо="Леся Українка"),
  жанр="драма"
)
```

```мавка
Тварина(
  вид="кіт",
  імʼя="Тимофій",
  вік=2,
  стать="чоловіча"
)
```

Записати створення числа можна так:

```мавка
1

-2

'ї'

3.14

-2.001
```

Записати створення тексту можна так:

```мавка
"я ю є"

"""Гетьте, думи, ви хмари осінні!"""
```

Записати створення списку можна так:

```мавка
[1, 3.14, -2, Людина(), дійсне, "ого!"]
```

Записати створення словника можна так:

```мавка
{"організація" => "діячі", "визначення" => "значення"}
```

Записати створення дії можна так:

```мавка
дія привіт()
  друк("привіт!")
кінець
```

```мавка
ще_привіт = дія()
  друк("ще привіт!")
кінець
```

```мавка
і_ще_привіт = дія(): друк("і ще привіт!")
```

Записати створення класу можна так:

```мавка
клас Молекула
  атоми
кінець
```

Записати створення модуля можна так:

```мавка
модуль двигуни
  клас ІоннийДвигун
    ;; параметри
  кінець

  клас РеактивнийДвигун
    ;; параметри
  кінець

  дати ІоннийДвигун
  дати РеактивнийДвигун
кінець
```
