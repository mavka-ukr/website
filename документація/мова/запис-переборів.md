# Запис переборів

Записати перебір в Мавці можна так:

```мавка
перебрати 0..10 як ел
  друк(ел)
кінець
```

```мавка
перебрати 0..=10 як ел
  друк(ел)
кінець
```

```мавка
перебрати [1, 3, 6] як ел
  друк(ел)
кінець
```

Назву для призначення елемента перебору можна не вказувати. В такому випадку її буде додано автоматично як `ел`:

```мавка
перебрати [1, 3, 6]
  друк(ел)
кінець
```

`перебрати`, по-суті, є просто надбудовою над `поки`:

```мавка
значення = [1, 3, 6]

перебір = значення.чародія_перебір()

поки перебір.далі()
  ел = перебір.значення

  друк(ел)
кінець

ел = недійсне
```

Як і `поки`, також доступні вказівки `перервати` і `перестрибнути`:

```мавка
перебрати [1, 3, 6] як ел
  якщо ел == 3
    перервати
  кінець

  друк(ел)
кінець
```

```мавка
перебрати [1, 3, 6] як ел
  якщо ел == 3
    перестрибнути
  кінець

  друк(ел)
кінець
```

Існує короткий запис перебору:

```мавка
перебрати 0..10 як ел: друк(ел)
```
