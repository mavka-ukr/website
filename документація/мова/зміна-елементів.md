# Зміна елементів

Змінити елемент предмета в Мавці можна так:

```мавка
а[б] = "Що це?"
```

```мавка
а.б[г].д()[е] = "Що це?" ;; ...тощо
```

```мавка
місяці_весни = ["березень", "квітень", "червень"]

друк(місяці_весни[2]) ;; червень

місяці_весни[2] = "травень"

друк(місяці_весни[2]) ;; травень
```
