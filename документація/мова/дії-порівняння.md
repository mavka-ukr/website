# 袛褨褩 锌芯褉褨胁薪褟薪薪褟

袟邪锌懈褋 锌芯褉褨胁薪褟谢褜薪懈褏 写褨泄 薪邪写 锌褉械写屑械褌邪屑懈 胁褨写斜褍胁邪褦褌褜褋褟 褌邪泻:

袪褨胁薪芯:

```屑邪胁泻邪
邪 == 斜
```

袧械 褉褨胁薪芯:

```屑邪胁泻邪
邪 != 斜
```

袘褨谢褜褕械 (谢懈褕械 写谢褟 褔懈褋械谢):

```屑邪胁泻邪
邪 > 斜
```

袦械薪褕械 (谢懈褕械 写谢褟 褔懈褋械谢):

```屑邪胁泻邪
邪 < 斜
```

袧械 斜褨谢褜褕械 (谢懈褕械 写谢褟 褔懈褋械谢):

```屑邪胁泻邪
邪 <= 斜
```

袧械 屑械薪褕械 (谢懈褕械 写谢褟 褔懈褋械谢):

```屑邪胁泻邪
邪 >= 斜
```

袆 (锌械褉械胁褨褉懈褌懈 泻谢邪褋 锌褉械写屑械褌邪):

```屑邪胁泻邪
邪 褦 效懈褋谢芯
```

袧械 褦:

```屑邪胁泻邪
邪 薪械 褦 袦芯谢械泻褍谢邪
```
