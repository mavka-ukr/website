# Чародії

## `чародія`

Дія `чародія` здійснюється під час створення предмета певного класу.

```мавка
клас Ракета
  назва
кінець

дія Ракета.чародія(назва)
  я.назва = "Ракета %(назва)"
кінець

ракета = Ракета("Реакція")

друк(ракета) ;; Ракета(назва="Ракета Реакція")
```

## `чародія_перебір`

Дія `чародія_перебір` здійснюється для отримання перебору.

```мавка
клас Машина
  елементи
кінець

дія Машини.чародія_перебір()
  ;; тут необхідно вернути перебір
кінець

машини = Машини([
  ;; ...
])

перебрати машини як м
  ;; ...
кінець
```

## `чародія_число`

Дія `чародія_число` здійснюється для перетворення значення на число.

```мавка
друк(Число("123"))
```

## `чародія_текст`

Дія `чародія_текст` здійснюється для перетворення значення на текст.

```мавка
друк(Текст(123.456))
```
