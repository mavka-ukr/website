# Запис повторів

Записати повтор з умовою в Мавці можна так:

```мавка
п = 0

поки п < 10
  друк(п)

  п += 1
кінець
```

Для керування повтором існують такі вказівки: `перервати`, `перестрибнути`.

`перервати` дозволяє "перервати" поточний повтор і продовжити виконання програми:

```мавка
п = 0

поки п < 10
  якщо п == 5
    перервати
  кінець

  друк(п)

  п += 1
кінець

друк(п) ;; 5
```

`перестрибнути` дозволяє перестрибнути на початок повтору:

```мавка
п = 0

поки п < 10
  якщо п == 5
    п = 8

    перестрибнути
  кінець

  друк(п)

  п += 1
кінець

друк(п) ;; 10
```

Існує також короткий запис поки:

```мавка
п = 0

поки п < 10: п += 1

друк(п)
```
